# Flutter WhatsApp UI

A Flutter-based UI clone of WhatsApp created for learning and UI practice purposes.

## Features
- Chat list UI
- App bar and tab layout
- Material design components

## Technologies Used
- Flutter
- Dart

## Purpose
This project helps in understanding Flutter layouts and building real-world app interfaces.
